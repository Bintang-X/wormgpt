# 🐛 WormGPT - AI Terminal Interface

<div align="center">

![Python Version](https://img.shields.io/badge/Python-3.x-blue.svg)
![API](https://img.shields.io/badge/Powered_by-BintangAPI-orange.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

**WormGPT AI Terminal Interface** adalah antarmuka *Command Line* (CLI) interaktif untuk berinteraksi dengan AI secara mulus langsung dari terminalmu. Dibangun dengan estetika *hacker/cyberpunk* yang kental, UI responsif, dan animasi yang *smooth*.

</div>

---

## ✨ Fitur Utama

*   **🖥️ Estetika Terminal Premium:** Dilengkapi dengan ASCII Art dinamis, warna ANSI yang kaya, dan *box-drawing* UI yang rapi.
*   **💬 Multi-Chat Session Management:** Kelola percakapan dengan mudah. Buat chat baru, beralih ke riwayat chat sebelumnya, atau *reset* seluruh sesi langsung dari menu.
*   **⏳ Animasi & Efek Visual:** Terdapat efek *typewriting* seperti manusia mengetik, animasi *spinner loading* saat AI berpikir, dan *boot animation* yang keren.
*   **💻 Code Block Support:** Output kode dari AI akan dipisahkan dan diformat dengan jelas tanpa memotong struktur kode asli, sehingga mudah untuk di- *copy-paste*.
*   **📡 Powered by BintangAPI:** Terintegrasi langsung dengan endpoint BintangAPI yang cepat dan stabil.

## 🚀 Prasyarat

Pastikan sistem kamu sudah terinstal:
*   [Python 3.x](https://www.python.org/downloads/)
*   Koneksi internet yang stabil

## 🛠️ Instalasi

1. **Clone repositori ini** ke dalam *local machine* kamu:
   ```bash
   git clone [https://github.com/USERNAME_GITHUB_KAMU/wormgpt-cli.git](https://github.com/USERNAME_GITHUB_KAMU/wormgpt-cli.git)
   cd wormgpt-cli
   

```
 2. **Install library yang dibutuhkan** (Script ini sebagian besar menggunakan library bawaan Python, hanya membutuhkan requests):
   ```bash
   pip install requests
   
   
   ```
```

3. **Jalankan aplikasi:**
   ```bash
   python3 main.py
   

```
*(Ganti main.py dengan nama file Python-mu jika berbeda).*
## 🎮 Cara Penggunaan
Setelah masuk ke dalam aplikasi, kamu akan disambut oleh *boot animation* dan menu interaktif.
 * **Ketik pesanmu** di prompt ┌─[ Masukkan Pesan ]─> lalu tekan **Enter**.
 * Ketik **menu** atau **help** untuk membuka panel opsi:
   * 1 - Lanjut Chat
   * 2 - Bikin Chat Baru (*New Session*)
   * 3 - Pindah Chat (*Switch History*)
   * 4 - Cek Info Session & Daftar Chat
   * 5 - Reset Semua Chat
   * 6 - Bersihkan Layar (*Clear*)
   * 7 - Keluar
 * Ketik **exit**, **quit**, atau tekan Ctrl+C untuk menutup terminal.
